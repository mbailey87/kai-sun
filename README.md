# kai-sun

Design repository for Kai+Sun. See `docs/information-architecture.md` for the current information architecture, layout options, and design system notes.

## Running the concept site

1. From the repo root, start a local server: `python -m http.server 8000`.
2. Open `http://localhost:8000` in your browser to explore the Kai+Sun landing page.
3. Use the filter buttons in **Menu highlights** to view different drink categories.
